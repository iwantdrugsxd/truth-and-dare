## Getting Started

### Prerequisites

- Flutter SDK (3.0.0 or higher)
- Dart SDK (3.0.0 or higher)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/partizo.git
cd partizo
```

2. Install dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
flutter run
```

## Game Flow

1. **Home Screen** - Start the game or view instructions
2. **Player Setup** - Add 2-8 players with unique names
3. **Spin the Bottle** - Animated spinner selects the next player
4. **Choose Your Fate** - Select Truth or Dare with optional timer
5. **Question Display** - View the challenge
6. **Timer Screen** - Complete the challenge within the time limit
